Welcome to River Bank

This is not a real bank.

This site is a mock web application of non-existent bank. It implements basic banking operations, such as deposit, withdraw and transfer. Moreover, all operations are saved and can be browsed anytime. To get started click sign up.

If you don't want to get through registration, you may use testing account.

Used technologies: Java, Spring Framework(MVC, Boot, Security, Data JPA), Hibernate ORM, PostgreSQL, HTML/CSS, Thymeleaf, Bootstrap.

For more information, please check GitHub.
